# Face Mask Classification Web Application

## Deskripsi
Proyek ini merupakan implementasi Deep Learning menggunakan metode Convolutional Neural Network (CNN) untuk mengklasifikasikan penggunaan masker pada wajah.

Aplikasi berbasis web ini memungkinkan pengguna mengunggah (upload) gambar, kemudian sistem akan melakukan deteksi wajah dan mengklasifikasikan apakah wajah tersebut menggunakan masker (With Mask) atau tidak menggunakan masker (Without Mask).


## Tujuan Proyek
Mengimplementasikan model CNN ke dalam aplikasi web untuk melakukan klasifikasi penggunaan masker secara otomatis berdasarkan gambar yang diunggah pengguna.
